1. 동기와 비동기

동기 : 여러개의 작업을 순서대로, 하나씩 처리하는 방식
Js는 동기적으로 코드를 실행시킨다.


P : 엔진에 쓰레드가 1개밖에 없기 때문. -> 비효율적
S : 비동기적이게 실행할 수 있도록 Web APIs라는 영역을 둠.

비동기 : 작업을 순서대로 처리하지 않음.

2. 비동기 처리 방식

J: JavaScript Engine
W : Web APIs 
라고 가정

J : 비동기함수 발견. W에게 실행 요청. 
비동기함수가 가지고있는 타이머, 콜백함수를 함께 넘김

W : 타이머 잘 보고있다가 시간 다되면 콜백함수 넘겨줌
